# دليل إضافة الأيقونة

## المشكلة التي تم حلها

تم حل مشكلة بناء البرنامج بإزالة تحديد الأيقونة من `package.json` مؤقتاً.

## إذا أردت إضافة أيقونة مخصصة

### 1. إنشاء ملف الأيقونة

يجب أن تكون الأيقونة بصيغة **PNG** أو **ICO** وليس SVG.

**المتطلبات:**
- **Windows**: ملف `.ico` أو `.png` بحجم 256x256 بكسل أو أكبر
- **macOS**: ملف `.icns` أو `.png` بحجم 512x512 بكسل
- **Linux**: ملف `.png` بحجم 512x512 بكسل

### 2. إنشاء الأيقونة أونلاين

يمكنك إنشاء أيقونة مجانية من المواقع التالية:
- [favicon.io](https://favicon.io/) - لإنشاء أيقونات بسيطة
- [icoconvert.com](https://icoconvert.com/) - لتحويل PNG إلى ICO
- [cloudconvert.com](https://cloudconvert.com/) - لتحويل بين جميع الصيغ

### 3. إضافة الأيقونة للبرنامج

بعد إنشاء ملف الأيقونة:

1. احفظ ملف الأيقونة في مجلد `public/` بالاسم `icon.png` (أو `icon.ico` لـ Windows)

2. افتح ملف `package.json`

3. أضف السطر التالي تحت قسم `"win"`:
```json
"win": {
  "target": [...],
  "icon": "public/icon.ico"
},
```

4. وأيضاً تحت قسم `"mac"` و `"linux"`:
```json
"mac": {
  "target": [...],
  "category": "...",
  "icon": "public/icon.png"
},
"linux": {
  "target": [...],
  "category": "...",
  "icon": "public/icon.png"
}
```

### 4. بناء البرنامج مرة أخرى

```bash
npm run electron:build:win
```

---

## ملاحظات مهمة

- **لا تستخدم ملفات SVG** - electron-builder لا يدعمها
- استخدم صور عالية الجودة (256x256 على الأقل)
- يفضل استخدام صور شفافة (PNG مع قناة alpha)
- تأكد من أن الأيقونة واضحة حتى بحجم صغير (16x16 بكسل)

---

## البناء بدون أيقونة

البرنامج الآن يُبنى بنجاح **بدون أيقونة مخصصة**. سيستخدم Electron الأيقونة الافتراضية تلقائياً.
